## Prerrequisitos
- Node.js
- npm
- Cuenta en Firebase
- Git

## Configuración del proyecto

- git clone https://github.com/Trikzster1/Consultoria-Grupo5
- cd Consultoria-Grupo5/kleoproject

## Instalar y ejecutar Firebase

- npm install -g firebase-tools
- firebase login
- firebase use audiosgrupo5-86b4d

## Configurar Variables de entorno

* crear archivo .env dentro de carpeta raíz del proyecto **kleoproject**

## Instalar Dependencias
Dentro de la carpeta raíz del proyecto **kleoproject** ejecutar:
- npm install
Luego instalar dependencias de backend **Functions**
- Moverse a carpeta **functions** con el comando **cd functions**
- npm install
- cd ( regresar a carpeta raíz )

## Correr proyecto
En carpeta raíz del proyecto **kleoproject** ejecutar:
- npm start
